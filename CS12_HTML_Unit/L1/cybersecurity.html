MySpace Injection

MySpace profiles used HTML markup for their profile pages. While this was originally fine, Samy Kamkar realized that by inserting a script tag, He could inject Javascript into the browser of anyone who opened his profile. He used this Javascript to cause users who opened his profile to add his profile, and copy the Javascript into their own profile. Within 20 hours, over one millon people had added him as a friend. 6 Months later he was arrested.

Heartbleed - Open SSL

The Heartbleed bug was a catastrophic vulnerability in the OpenSSL library that allowed attackers to "bleed" sensitive data from a server's memory. It exploited a flaw in the TLS Heartbeat extension, where an attacker could send a malformed request claiming to have a large amount of data; the server, failing to verify the actual size of the payload, would respond by copying up to 64KB of its own memory to fill the request. Because this memory often contained private encryption keys, usernames, and passwords, Heartbleed essentially broke the fundamental trust of the secure internet. Most chillingly, the exploit left no trace in system logs, making it impossible to know for sure how much data was stolen before the fix was deployed in 2014.
